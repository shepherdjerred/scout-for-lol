# Report UI

This package presents a browser-compatible version of the report package using React and Vite.

It's useful for iterating on the report design without having to re-render static images.

## Development

```bash
# Start development server (runs on port 5173)
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview

# Run tests
bun run test

# Type check
bun run typecheck

# Lint
bun run lint

# Format code
bun run format:write
```
